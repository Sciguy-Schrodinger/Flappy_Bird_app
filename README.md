# 🐦 Flappy Bird

A classic Flappy Bird clone built with **Pygame**.  
Navigate the bird through randomly generated pipes and try to beat your high score!

---

## 🎮 Features

- **Classic Flappy Bird gameplay** — tap to flap, avoid pipes
- **Randomized pipe generation** — pipes appear at random positions and speeds
- **Score tracking** — earn points for each pipe you pass
- **Dynamic pipe sizes** — different pipe heights for varied difficulty
- **Sound effects** — "ding" sound when scoring
- **Simple controls** — just one key to play!

---

## 🛠️ Technologies Used

- **Python 3** — core logic
- **Pygame** — 2D rendering, input handling, audio
